
# SMR WP-Plugin

Example for WordPress plugin in PHP.
<small>version 0.1</small>

## How to use
- clone repository.
- copy to plugin folder.
- activate it.
- good luck.

## Dependencies
- PHP Composer.

## Tested On
- [PHP v7.4.0](https://www.php.net/releases/7_4_0.php)
- [WordPress v5.7.0](https://wordpress.org/download/releases/)
